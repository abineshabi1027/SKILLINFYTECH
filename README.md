💡 About the Repository
This repository contains a series of data-driven projects focused on solving real-world business problems. From analyzing customer churn and evaluating digital product performance to uncovering workforce trends, these scripts utilize standard data science workflows to transform raw datasets into strategic insights.

🛠️ Tech Stack
Language: Python 3.x

Data Manipulation: pandas, numpy

Data Visualization: matplotlib, seaborn

Environment: Jupyter Notebook / Local IDE / Google Colab

📂 Projects Overview
1. Customer Retention Analysis
Goal: Gain insights into customer lifetime value and improve business retention strategies.

What it does: Segments customer purchasing behavior by Recency, Frequency, and Monetary value (RFM) to identify high-value users and critical churn factors.

Concepts & Skills Learned:

RFM Analysis

Cohort Segmentation

Correlation Analysis

pandas GroupBy operations and Pivot Tables

2. Web Traffic Analytics
Goal: Measure digital product performance and identify opportunities to improve UX and conversion rates.

What it does: Processes website traffic data to map out user behavior, identifying drop-off points, session paths, and top-performing pages.

Concepts & Skills Learned:

Conversion Funnel Analysis

Session and Page Path tracking

Visualizing User Behavior

Identifying UX friction points

3. HR Attrition Analysis
Goal: Use analytics in HR to enhance workforce planning and proactively reduce employee turnover.

What it does: Performs deep statistical analysis on human resources datasets to uncover underlying patterns related to age, job role, salary, and company tenure.

Concepts & Skills Learned:

Descriptive Statistics and Histograms

Crosstab and Correlation Heatmaps

Attrition Prediction Trends

Insight Storytelling with Visuals
